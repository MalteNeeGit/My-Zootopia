# 🐾 Animal Information Generator

Dieses Projekt ist ein Python-basiertes Tool, das Informationen über verschiedene Tierarten von einer externen API abruft und daraus eine visuell ansprechende Webseite generiert.

## 📖 Projektbeschreibung
Der "Animal Information Generator" ermöglicht es Nutzern, nach einem Tiernamen zu suchen. Das Programm kommuniziert mit der [API Ninjas Animals API](https://api-ninjas.com/api/animals), um wissenschaftliche Daten wie Ernährung, Standorte und Tierarten abzurufen. Diese Daten werden anschließend in ein HTML-Template eingebettet, um eine übersichtliche "Animal Repository"-Webseite zu erstellen.
